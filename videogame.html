2D Videogame Development

 One of my works in progress is a 2D hack-and-slash type of game which would allow for numerous interactions with enemies and NPCs across the map. The ideal aspect of the game would be for a story-based game with a seemingly linear progression.

 All artwork demonstrated in the video (located on the left) was created by me using Aseprite. The game was coded using GameMaker Studio 2 and the GameMaker language.

3D Videogame Development

 My other work involves learning 3D videogame design, specifically using the Unity Engine. Specifically, I am trying to create a souls-like game, or at least begin learning to create a videogame based on that exact premise. As can be expected, this comes from my enjoyment of many souls-like games and I hope to be able to expand my learning in game development from the tutorial series that I have been watching.

 The tutorial series created by Sebastian Graves is an amazing tutorial series for anyone looking to get into Unity or game development. I would suggest, as Sebastian does, to learn a little bit about the basics of Unity and general computer science knowledge before starting this series, but it's a great series to go with.
