Harry's Heroes

With the help of the Harry's Heroes Organization, at 501(c)3 non-profit, I am setting up an event at Jersey Shore Arena on April 23rd. This event will help raise money to help fight against muscular dystrophy and other diseases.

Participants in the event will be able to play 2 hockey games. In order to guarantee a spot, since there can only be so many participants, if you send a donation of $25.00, we will guarantee that you will play. The donations will also help the Harry's Heroes Organization, so it is going to a good cause. Please view the flyers for more information and we hope to see you there!
